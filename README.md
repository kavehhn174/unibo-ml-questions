# Unibo ML Questions

Practice material for the University of Bologna Machine Learning & Data Mining (MLDM) exams.

## Contents

- **[`MLDM Exams.md`](./MLDM%20Exams.md)** — raw compilation of past exam questions and answers (Simulation 2022, Written Exam 2022-02-07, and the 2020-25 comprehensive guide).
- **[`quiz-app/`](./quiz-app)** — an interactive Vue 3 + PrimeVue quiz app built from that material: 68 multiple-choice questions with explanations, dark/light theme, bookmarking, and 1–5 star importance ratings. See [`quiz-app/README.md`](./quiz-app/README.md) for full details.

## Quick start

```bash
cd quiz-app
npm install
npm run dev
```

Or just double-click `quiz-app/run.sh`.
